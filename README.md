# my-static-website
Overview

This project is a static website hosted on AWS using a serverless architecture. Amazon S3 stores and serves the website files, while Amazon CloudFront distributes the content globally for improved performance and low latency. A CI/CD pipeline using GitHub Actions automatically deploys updates to S3 whenever changes are pushed to the repository. HTTPS is enabled to ensure secure access to the website.

This repository contains the frontend code (index.html) and the CI/CD workflow configuration used for automated deployment.

Screenshots of the live website, S3 bucket, CloudFront distribution, and CI/CD pipeline are included to demonstrate successful deployment and system architecture.
